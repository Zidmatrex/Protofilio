# Abdulrahman Zidan Portfolio

A premium, responsive personal portfolio website for Abdulrahman Zidan, designed as a polished real-estate-focused personal brand site for GitHub Pages deployment.

## Features
- Modern dark theme with premium UI details
- Responsive sections for hero, about, experience, skills, software, and contact
- Smooth scrolling, reveal animations, animated counters, and typing effect
- SEO-ready meta tags, Open Graph, Twitter Cards, structured data, robots, and sitemap
- Static deployment compatible with GitHub Pages

## Files
- index.html
- style.css
- script.js
- assets/
- robots.txt
- sitemap.xml

## Local Preview
Open index.html directly in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000/.

## GitHub Pages
1. Upload these files to the root of your GitHub Pages repository.
2. Ensure the site is published from the main branch.
3. Replace the placeholder resume and profile image paths in the assets folder when you upload your own files.
